# dotfiles

Personal configuration files.

## Set up 

| Tool       | Config location              | Symlinked to               |
|------------|------------------------------|----------------------------|
| Neovim     | `nvim/`                      | `~/.config/nvim`           |
| Tmux       | `tmux/tmux.conf`             | `~/.tmux.conf`             |
| Alacritty  | `alacritty/alacritty.toml`   | `~/.config/alacritty`      |
